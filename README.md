# 3D Cube

3D rotating cube rendered in a Windows terminal using C++.

## Features

- 3D projection
- X/Y/Z rotation

## Preview

![Cube Demo](preview.gif)

## Compile

Using g++:

```bash
g++ src/main.cpp -o cube.exe -static -static-libgcc -static-libstdc++
```

## Requirements

- Windows
- C++11 or newer
